# Application Pipeline Starter

![forthebadge](http://forthebadge.com/images/badges/makes-people-smile.svg)

Boilerplate code that setups test - dev and production piplelines for web application
development.

### Technology Stack

##### Front End
* SystemJS
* Bootstrap
* jQuery
* Bluebird
* React
* ReactDOM

##### Transpiler
* Typescript

##### Bundler
* SystemJS Builder

##### Unit Testing
* Karma
* Mocha
* Chai
* Sinon

### Commands

* ```npm run build```
* ```npm run test```
* ```npm run dev```
* ```npm run prod```
