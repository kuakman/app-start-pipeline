# Application Pipeline Starter

![forthebadge](http://forthebadge.com/images/badges/makes-people-smile.svg)

Boilerplate code that setups test - dev and production pipelines for web application development.

### Technology Stack

##### Front End
* SystemJS
* Bluebird
* Bootstrap
* jQuery
* React
* ReactDOM
* React-Logger
* Redux
* Redux-Actions
* Redux-Promise
* Redux Socket.io
* Socket.io

##### Transpiler
* Typescript

##### Bundler
* SystemJS JSPM Builder

##### Unit Testing
* Karma
* Mocha
* Chai
* Sinon

### Commands

* ```npm run build```
* ```npm run test```
* ```npm run dev```
* ```npm run prod```
